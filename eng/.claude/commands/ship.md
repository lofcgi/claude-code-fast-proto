Deploy the app in project/.

## Tasks

1. **Verify deployment service CLI**
   - Read `tech_stack.deploy` from `analysis/requirements.json`
   - Check CLI installation:
     - Vercel → `npx vercel --version` check
     - Netlify → `npx netlify --version` check
     - Cloudflare → `npx wrangler --version` check
     - Other → "Please install the CLI for this service: [install link]" then stop

2. **Collect environment variables**
   - Read `project/.env.example` (generated by implement)
   - Ask user for each key's value
   - Create `project/.env.local` with provided values
   - Set env vars on deployment service (CLI commands)

3. **Pre-deploy build check**
   - `cd project/ && npm run build`
   - Fix and retry on failure

4. **CLI deployment**
   - Vercel: `cd project/ && npx vercel --prod`
   - Netlify: `cd project/ && npx netlify deploy --prod`
   - Other: appropriate CLI command

5. **Result summary**
   - Print deployment URL
   - Guide additional setup if needed (custom domain, DNS, OAuth redirect URI, etc.)
