# 기술 제약사항 분석

## ElevenLabs API
- **무료 티어**: 월 10,000자 전사, 제한된 TTS 크레딧
- **파일 크기**: 최대 25MB (무료)
- **지원 포맷**: MP3, MP4, WAV, M4A, WEBM
- **Rate Limit**: 분당 제한 존재
- **영향**: 대용량/긴 파일은 처리 불가, 사용량 추적 필요

## Turso (libSQL)
- **무료 티어**: 9GB 스토리지, 500M 행 읽기/월
- **연결 수**: 3개 데이터베이스
- **영향**: 대량 사용자/히스토리 시 스토리지 부족 가능

## Vercel 배포
- **서버리스 함수**: 무료 티어 최대 10초 실행 시간
- **대역폭**: 100GB/월
- **영향**: ElevenLabs API 호출이 10초 초과 시 타임아웃 → 스트리밍 또는 비동기 처리 필요
- **해결**: API Route에서 streaming response 또는 클라이언트 직접 호출

## Auth.js v5
- **Google OAuth**: Google Cloud Console에서 OAuth 클라이언트 ID 발급 필요
- **콜백 URL**: Vercel 배포 도메인 설정 필요
- **제약**: 로컬 개발 시 localhost 콜백 별도 설정

## 파일 저장
- **Vercel**: 영구 파일 저장 미지원 (서버리스)
- **해결**: 결과 파일은 임시 URL로 제공하거나, 외부 스토리지 (R2, S3) 연동
- **프로토타입**: 파일을 메모리/임시 경로에서 처리 후 즉시 반환
