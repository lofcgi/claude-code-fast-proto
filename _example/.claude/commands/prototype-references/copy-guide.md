# Copy Writing Guide (카피 작성 공식)

프로토타입의 모든 텍스트 콘텐츠는 이 가이드를 따라야 한다.
"AI가 쓴 냄새"를 제거하고 프로 카피라이터 수준의 콘텐츠를 생성하라.

---

## 헤드라인 공식

**공식**: [동사] + [구체 대상] + [수치 혜택]

| Bad (AI Slop) | Good (Pro Copy) | 왜 좋은가 |
|---------------|-----------------|----------|
| "혁신적인 AI 더빙 솔루션" | "33개 언어, 4.2초" | 구체적 수치가 신뢰감 |
| "차세대 프로젝트 관리" | "Ship 2x faster" | 행동 + 수치 혜택 |
| "올인원 마케팅 플랫폼" | "From draft to revenue in 47 minutes" | 구체적 시간 약속 |
| "강력한 데이터 분석" | "Find the signal in 2M rows" | 구체적 규모 + 행동 |

### 금지어 (AI Slop 마커)
- 혁신적인, 차세대, 올인원, 강력한, 최첨단
- Revolutionary, Next-gen, All-in-one, Powerful, Cutting-edge
- Seamlessly, Effortlessly, Unlock, Leverage, Streamline
- "당신의 ~를 혁신합니다"

### 권장 패턴
- **수치 리드**: "98.5% uptime. Zero excuses."
- **대비법**: "Less meetings. More shipping."
- **직접법**: "Turn recordings into text. Instantly."
- **브랜드 동사**: 제품 고유의 동사 창조 (Notion: "wikify", Stripe: "accept")

## Feature 이름 공식

**공식**: [브랜드 동사] + [능력]

| Bad (일반 명사) | Good (브랜드 용어) | 규칙 |
|----------------|------------------|------|
| "음성 합성" | "VoxSync Engine" | 고유명사 + 기술 용어 |
| "자동 번역" | "LinguaFlow" | 합성어 브랜드명 |
| "실시간 분석" | "PulseMetrics" | 감각적 + 기능적 |
| "협업 도구" | "TeamForge" | 행동 + 은유 |
| "AI 어시스턴트" | "CoPilot Zero" | 브랜드 + 버전감 |

### Feature 설명 공식
- **1문장**: [무엇을] + [어떻게] + [결과]
- **15-20 단어 제한**
- **구체적 수치 1개 포함**

| Bad | Good |
|-----|------|
| "AI를 활용한 강력한 음성 합성 기능" | "33개 언어로 4.2초 만에 자연스러운 음성을 생성합니다" |
| "효율적인 프로젝트 관리" | "드래그 한 번으로 142명 팀의 태스크를 자동 재배치" |

## CTA 공식

**공식**: [행동] + [즉각 혜택]

| Bad (기본) | Good (행동+혜택) | 규칙 |
|-----------|-----------------|------|
| "시작하기" | "첫 10분 무료 체험" | 시간 한정 혜택 |
| "가입하기" | "30초만에 시작" | 속도 약속 |
| "무료 체험" | "카드 없이 시작" | 장벽 제거 |
| "더 알아보기" | "데모 영상 보기" | 구체적 행동 |
| "Get Started" | "Start building free" | 행동 + 무료 |
| "Sign Up" | "Create your first project" | 구체적 첫 단계 |
| "Try Now" | "See it in action — 2 min demo" | 시간 약속 + 행동 |

### CTA 배치 규칙
- **Primary CTA**: glow box-shadow, 큰 크기 (px-10 py-4)
- **Secondary CTA**: ghost/outline, Primary 옆에 배치
- **Hero**: 반드시 2개 CTA (Primary + Secondary)
- **Footer CTA**: 반드시 1개 이상 CTA
- **Floating CTA**: 스크롤 진행도에 따라 하단 고정 CTA 고려

## 수치 공식

**규칙**: 구체적 > 둥근 > 없는 것

| Bad (둥근) | Good (구체적) | 규칙 |
|-----------|--------------|------|
| 10,000+ | 10,847 | 홀수가 더 믿음직 |
| 100% | 99.97% | 완벽보다 구체적 |
| 99% | 98.5% | 소수점이 신뢰감 |
| 1M | 1.2M | 소수점 추가 |
| 2x faster | 2.3x faster | 소수점 추가 |
| 50+ countries | 47 countries | 정확한 숫자 |
| Thousands | 4,281 | 정확한 숫자 |

### Trust Metrics 섹션 수치 예시
```
10,847 → "Active teams"
98.5% → "Uptime SLA"
4.2s → "Average response"
47 → "Countries served"
```

### 수치 사용 위치
1. **Trust Metrics 섹션**: NumberTicker로 count-up 4개
2. **Feature 설명**: 각 Feature에 1개씩
3. **헤드라인**: 가능하면 1개
4. **CTA 근처**: Social proof 수치 1개
5. **Footer**: "Join X teams" 형태

## 소셜 프루프 공식

### Testimonial 구조
```
"{구체적 결과} — {제품명} 덕분에 {구체적 시간/비용 절감}"
— {이름}, {구체적 직함} @ {회사명}
```

| Bad | Good |
|-----|------|
| "훌륭한 제품입니다" | "온보딩 시간이 3주에서 4일로 줄었습니다" |
| "매우 만족합니다" | "매달 $12K 절감 — 6개월 ROI 달성" |
| "추천합니다" | "142명 팀이 첫 주에 자발적으로 전환했습니다" |

### 로고 바 (Trust Bar)
- 실제 브랜드 로고 사용 불가 시: "Trusted by teams at companies like..."
- grayscale → hover:color 효과
- InfiniteMarquee로 움직임 추가

## 톤 & 보이스 매트릭스

| 프로젝트 타입 | 톤 | 헤드라인 스타일 | 예시 |
|-------------|-----|---------------|------|
| B2B SaaS | 자신감, 간결 | 수치 리드 | "Ship 2x faster" |
| AI/ML 도구 | 기술적, 정확 | 능력 리드 | "Process 2M records in 4.2s" |
| 크리에이티브 도구 | 영감, 대담 | 감각 리드 | "Design at the speed of thought" |
| 생산성 앱 | 실용적, 직접적 | 문제 리드 | "Never miss a deadline again" |
| 금융/핀테크 | 신뢰, 안정 | 안전 리드 | "99.97% uptime. Bank-grade security." |
| 교육/EdTech | 친근, 지지적 | 변화 리드 | "From beginner to builder in 30 days" |

## 체크리스트 (카피 최종 검증)

- □ 헤드라인에 금지어 0개
- □ 모든 수치가 구체적 (둥근 숫자 0개)
- □ Feature 이름이 브랜드 고유 용어
- □ CTA가 동사+혜택 형태
- □ Testimonial에 구체적 결과 포함
- □ 모든 설명이 20단어 이하
- □ 1개 이상의 Feature에 수치 포함
